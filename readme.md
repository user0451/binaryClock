# Binary Clock

This page is a live, animated binary clock that shows the current time with glowing bubbles.
It is designed to be both visual and educational, so you can switch between simple viewing and learning mode.

## What You See
- Three columns for Hours, Minutes, and Seconds.
- Animated bubbles that turn on/off to represent each value.
- A digital companion panel (optional) that shows H / M / S values and AM/PM in 12-hour format.

## How To Open Settings
- On larger screens, settings appear as a control bar near the bottom.
- On smaller screens, use the **Settings** button to open the controls panel.

## Features

### 1) 4-bit / 6-bit Mode
- **6-bit**: each column shows a full binary value for Hours, Minutes, and Seconds.
- **4-bit**: each value is split into Tens and Units for easier decimal reading.
- Switching modes includes a movement animation for a smooth, cinematic transition.

### 2) 12-hour / 24-hour Time
- Toggle between 24-hour and 12-hour formats.
- In 12-hour mode, the digital panel shows a small AM/PM badge.

### 3) Help Mode
- Overlays row values directly on the bubbles.
- Shows plus markers between rows to make the math easier to follow.
- Useful when learning how binary values add up.

### 4) Digital Panel Toggle
- Turn the digital panel on or off independently of Help Mode.
- The panel includes H / M / S values with rolling digit animation.
- Tens and units animate independently for a dial-like feel.

### 5) Theme Picker
- Choose from five visual themes:
	- Classic RGB Neon
	- Amber Terminal
	- Ice Glass
	- Matrix Pulse
	- Sunset Bloom
- Theme changes include coordinated color updates across the clock, help markers, and digital panel.

## Quality of Life
- Your settings are remembered automatically between visits.
- Works across desktop and mobile layouts.
- Includes reduced-motion support for users who prefer less animation.

## Tips
- If you are new to binary, start with **4-bit + Help**.
- If you want a clean display, use **6-bit + Digital panel on + Help off**.
- Try theme switching while running to see animated transitions.
